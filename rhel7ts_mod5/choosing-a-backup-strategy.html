5.3. Choosing a Backup Strategy

Choosing the right solution is up to the individual organization’s needs, budget, and expertise.
A good enterprise backup solution for a RHEL environment should:
  • Be able to support RHEL without much effort
  • Be a client/server architecture so that many systems can be backed up with one centralized solution
Nice to haves may include:
  • Support for multiple platforms so that one solution will work across all supported OS’s in an enterprise
  • Provide an API or command line interface for scripting
  • Be able to off-line media and track that media
