5.4. Backups And Virtualization

With the introduction of cloud computing and virtualization, traditional backup methodology has to be rethought.
  • Virtual systems are no longer tied to hardware, thus bare metal restoration is no longer required. Bare metal systems need only be able to run as hypervisors without any stateful data stored on them.
  • A virtual system is basically a file on disk so a myriad of new methods of creating full backups of entire systems is now available.
    • Real time replication of entire systems is now possible.
    • Entire systems can fail between physical locations without a restore.
